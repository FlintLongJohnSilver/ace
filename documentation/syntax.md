# Syntax

## Indent

A unit of an indent must be 2 spaces.

```ace
html
  body
    div
    p
```

becomes

```html
<html>
  <body>
    <div></div>
    <p></p>
  </body>
</html>
```

## HTML Tags

A line head is interpreted as an HTML tag.

```ace
div
  address
  i
  strong
```

becomes

```html
<div>
  <address></address>
  <i></i>
  <strong></strong>
</div>
```
